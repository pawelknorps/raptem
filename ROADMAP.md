# Raptem Development Roadmap (2026)

## Phase 1: The Soul of the Machine (Q2 2026)
*Focus: RPG Mechanics & Persistence*
- **Persistent Player Stats**: Implementation of `Folk Purity`, `Wixa Mana`, and `Chaos Affinity`.
- **Character Profile App**: A centralized hub to view player progress and achievements.
- **Supabase Sync**: Real-time persistence of inventory and RPG stats.
- **Interaction Rewards**: XP gains for using the Accordion, Terminal, and Scanner.

## Phase 2: The Living Legend (Q3 2026)
*Focus: Narrative & World Building*
- **Dynamic Quests**: Branching storylines with regional NPCs (Mirek, Janek).
- **World States**: The OS environment changes based on player choices (e.g., Ritual purity vs Chaos).
- **Regional Anomalies**: Time-limited events on the Kujawy Map.
- **Mirek AI v2**: Enhanced chat interaction with quest-awareness.

## Phase 3: The Grand Spectacle (Q4 2026)
*Focus: Media & Global Interaction*
- **Procedural Theater**: AI-assisted script generation for virtual performances.
- **Global Bazar**: Multi-player features (Guestbook, Shared Artifacts).
- **Performance Excellence**: Sub-100ms interaction latency on mobile.
- **VR/3D Expansion**: Immersive "Teatr Prędkości" 3D showroom.
